# Exam Stats Graph Q2A Plugin

Displays exam question statistics with graphs based on difficulty, subject, and type 

## Installation

1. Copy to `qa-plugin/exam-stats-graph` folder
2. Enable in Admin → Plugins
3. Configure the widget in Admin → Layout → Exam Question Statistics Graph
